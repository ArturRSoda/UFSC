Exercício com Mutmut - Python
❑

❑

Instale a ferramenta (version 3.x):
pip install mutmut

Para configurar o código que será mutado e os testes que
serão executados, crie o arquivo pyproject.toml no diretório
raiz do projeto com os seguintes comandos, por exemplo:
paths_to_mutate = ["src/puzzle_game.py"]
tests_dir = ["tests/"]

❑

Para executar o Mutmut a partir do diretório raiz:
PYTHONPATH=src:tests mutmut run

Exercício com Mutmut - Python
❑

Para visualizar os resultados:
mutmut results
mutmut show <mutant_id>

Ou

mutmut browse

Website: https://pypi.org/project/mutmut/
